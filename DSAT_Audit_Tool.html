Google Sheets Connection
How to connect (one-time setup):
1. Go to sheets.google.com → create a new blank sheet named "DSAT Audit Data"
2. Click Extensions → Apps Script → delete all code
3. Paste the script from the file google_apps_script.js (in your Dsat Tool folder)
4. Click Deploy → New deployment → Web app
5. Set "Execute as: Me" and "Who has access: Anyone" → Deploy
6. Copy the web app URL and paste it below
Paste the URL from Step 6 above. Once saved, all QA audits will automatically go to your Google Sheet.
